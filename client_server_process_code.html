C# TCP Network Process

Visual Guide to Server Socket Creation, Binding, Listening, and Client Connections

SERVER SIDE
1
Create Socket
Initialize a new TCP socket using the Socket class. Specify address family (IPv4), socket type (Stream for TCP), and protocol type.
Socket serverSocket = new Socket( AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.Tcp );
2
Bind to Endpoint
Bind the socket to a local IP address and port number. This associates the socket with a specific network interface and port.
IPEndPoint endPoint = new IPEndPoint( IPAddress.Parse("127.0.0.1"), 8080 // Port number ); serverSocket.Bind(endPoint);
3
Listen for Connections
Put the socket in listening mode. The backlog parameter specifies the maximum queue length for pending connections.
serverSocket.Listen(10); // Backlog of 10 Console.WriteLine("Server listening...");
WAITING FOR CONNECTIONS
4
Accept Client Connection
Block and wait for an incoming client connection. Returns a new Socket object representing the client connection.
Socket clientSocket = serverSocket.Accept(); Console.WriteLine("Client connected!");
BLOCKING CALL
5
Send/Receive Data
Exchange data with the connected client using Send() and Receive() methods on the client socket.
// Receive data byte[] buffer = new byte[1024]; int bytesRead = clientSocket.Receive(buffer); // Send response byte[] response = Encoding.UTF8.GetBytes("Hello!"); clientSocket.Send(response);
6
Close Connection
Gracefully close the client socket connection and optionally the server socket.
clientSocket.Shutdown(SocketShutdown.Both); clientSocket.Close(); // serverSocket.Close(); // When done
CLIENT SIDE
1
Create Socket
Initialize a client socket with the same parameters as the server socket.
Socket clientSocket = new Socket( AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.Tcp );
2
Connect to Server
Establish a connection to the server by specifying the server's IP address and port number.
IPEndPoint serverEndPoint = new IPEndPoint( IPAddress.Parse("127.0.0.1"), 8080 ); clientSocket.Connect(serverEndPoint); Console.WriteLine("Connected to server!");

3
Send/Receive Data
Send data to the server and receive responses using Send() and Receive() methods.
// Send data to server byte[] data = Encoding.UTF8.GetBytes("Hello Server!"); clientSocket.Send(data); // Receive response byte[] buffer = new byte[1024]; int bytesRead = clientSocket.Receive(buffer);
4
Close Connection
Gracefully shutdown and close the socket when communication is complete.
clientSocket.Shutdown(SocketShutdown.Both); clientSocket.Close();
